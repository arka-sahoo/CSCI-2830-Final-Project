# Student Management System

## This is a Student Management System which uses a JavaFX GUI and file processing.
## In the GUI, there are three input fields for student ID, student name, and student GPA.
## There are two buttons - Add and Remove.
## The Add button is used to add a student. The remove button is used to remove a student.
## There is a txt file in which the student details get added or removed according to the users input.
